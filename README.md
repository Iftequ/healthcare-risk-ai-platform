# 🏥 Healthcare Risk AI Platform

## 📌 Overview

This project builds an end-to-end machine learning and Generative AI system to predict patient risk, segment populations, and provide clinical decision support using structured and unstructured healthcare data.

## 🚀 Features

* Patient risk prediction using XGBoost / LightGBM
* Patient segmentation using clustering
* Model explainability using SHAP
* RAG-based clinical Q&A system using LLMs
* End-to-end ML pipeline with MLOps

## 🛠 Tech Stack

* Python, PySpark, SQL
* XGBoost, LightGBM
* MLflow, Docker, FastAPI
* AWS (S3, SageMaker)
* FAISS / Pinecone (RAG)

## 🎯 Use Case

Early detection of high-risk patients to reduce hospital readmissions and improve care management.
